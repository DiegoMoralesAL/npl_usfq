Predicción de Género de Canciones Usando Modelos de Aprendizaje Automático

Descripción del Proyecto

Este proyecto tiene como objetivo construir varios modelos de aprendizaje automático para predecir el género de una canción basándose en las letras transcritas de canciones obtenidas de YouTube. El proyecto utiliza técnicas de Procesamiento de Lenguaje Natural (NLP) para preprocesar las letras, así como diferentes modelos de clasificación para realizar la predicción de género.

Objetivo

El objetivo es crear un sistema que, dado el texto de una letra de canción, sea capaz de predecir su género utilizando varios modelos de clasificación. El sistema permite comparar modelos de diferentes versiones (v1 y v2) y analizar la distribución de predicciones con un enfoque en el rendimiento.

Datos Utilizados

Se utilizó una base de datos de canciones que incluye los siguientes campos:

  • artist_name, track_name, release_date, genre, lyrics, len, entre otros.

Las letras de las canciones fueron preprocesadas para eliminar stopwords, caracteres especiales y reducirlas a un formato útil para los modelos.

Modelos Implementados

Se entrenaron los siguientes modelos de clasificación para predecir el género de las canciones:

  • Naive Bayes (v1 y v2)
  • MLPClassifier (v1 y v2)
  • Random Forest (v1 y v2)
  • Logistic Regression (v1 y v2)
  • SVM (v1 y v2)
  • Gradient Boosting (v1 y v2)

Proceso de Preprocesamiento

El preprocesamiento incluyó:

  • Limpieza de Texto: Eliminación de caracteres especiales, stopwords, y conversión a minúsculas.
  • Vectorización: Utilización de TF-IDF para transformar las letras en vectores numéricos.

Resultados

Se realizaron predicciones utilizando fragmentos de las letras y las letras completas combinadas. Las predicciones de las letras completas recibieron un peso de x3 para darles mayor relevancia en el resultado final. Los modelos v1 y v2 se compararon en términos de precisión y consistencia.

Conclusiones

Los modelos v2 mejoraron significativamente en comparación con las versiones v1, especialmente en el caso de Random Forest y Gradient Boosting. Estos modelos demostraron una mayor precisión y consistencia en las predicciones, tanto para las letras separadas como para las letras completas combinadas.

Se observó una mayor variabilidad en los resultados del MLPClassifier, lo que indica que la estructura de la red neuronal podría beneficiarse de ajustes adicionales.
